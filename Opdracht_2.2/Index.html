HTML Fundamentals


Het element -h1 wordt gebruikt als een hoofdkop.

Het -p element staat voor paragraaf en wordt gebruikt on tekstinhoud te structeren.


Het -a element staat for "anker" en wordt gebruikt om hyperlinks tee maken naar andere webpaginas

Het -img element wordt gebruikt om afbeeldingen in een webpagine te voegen


Het -table element wordt gebruikt om een tabellen in in een webpagina te maken.

ul/ol/li elementen worden gebruikt om lijsten te maken.

-ul staat voor Unordered list

-ol staat voor Ordered list

-li staat voor list item
